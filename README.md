# Codsoft.java
CodSoft Java Internship Projects
This repository contains the Java tasks completed during my CodSoft UI/UX Internship. The projects focus on core Java programming concepts, logic building, and user interaction through console-based applications.
📌 Tasks Completed
1. Student Grade Calculator
A simple Java program that calculates the grade of a student based on marks obtained in different subjects.
Features:
Takes input for multiple subjects
Calculates total and average marks
Assigns grade based on percentage
Displays final result
2. Currency Converter
A console-based application that converts currency values between different currencies using predefined conversion rates.
Features:
Converts between multiple currencies
User-friendly input system
Accurate conversion using fixed rates
Easy to extend for more currencies
3. ATM Interface
A simulation of an ATM system that allows users to perform basic banking operations.
Features:
User authentication (PIN-based system)
Balance inquiry
Deposit and withdrawal functionality
Transaction validation and error handling
🛠️ Technologies Used
Java (Core Java)
OOP Concepts
Scanner Class for input handling
Console-based UI
📚 Learning Outcomes
Strengthened understanding of Java fundamentals
Improved problem-solving and logical thinking
Practical implementation of OOP concepts
Experience in building interactive console applications
